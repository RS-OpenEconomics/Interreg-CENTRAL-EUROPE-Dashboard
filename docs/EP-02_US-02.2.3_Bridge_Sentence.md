## US-02.2.3 — Normative Bridge Sentence

**Epic:** EP-02 — Need Description and Baseline Scenario
**Feature:** F-02.2 — Scenario Zero Narrative
**Priority:** Must Have
**Dependencies:** US-02.2.1 or US-02.2.2

### 1. What this screen/component does
Not a visible UI component. Defines a fixed text paragraph that is appended to the Scenario Zero narrative during DOCFAP report generation. The sentence is not shown in the wizard and is not editable by the user.

### 2. Purpose
Provide the structural bridge between the qualitative Scenario Zero section and the quantitative comparison of alternatives in the final DOCFAP report. This is a standard normative closing required by D.Lgs. 36/2023, Annex I.7.

### 3. Input data

None.

### 4. Pre-populated or default data

The sentence is hardcoded:

> "L'analisi dello stato di fatto e la proiezione dello scenario di non intervento evidenziano la necessità di procedere con l'investimento. Le alternative progettuali vengono di seguito confrontate quantitativamente per individuare la soluzione ottimale."

### 5. Business rules and validations
- The sentence is fixed and not configurable per need or per theme.
- It is not stored in `scenarioZeroNarrative` in the wizard store.
- It is appended by the report generation engine at render time, after the assembled or free-text narrative.
- It is not modifiable by the user.

### 6. UI element behaviour

None — this US defines report-generation behaviour, not a wizard UI element.

### 7. Frontend → OpenCore flow

```
Report generation triggered (Phase 5)
  → Engine reads scenarioZeroNarrative from store
  → Appends the fixed bridge sentence
  → Outputs the complete Scenario Zero section
```

### 8. OpenCore entities involved

None.

### 9. Edge cases and conditional behaviour

None.

### 10. Acceptance Criteria

```gherkin
Scenario: Bridge sentence appended in report
  Given the DOCFAP report is being generated
  When the Scenario Zero section is assembled
  Then the fixed bridge sentence is appended after the narrative text
  And it is not editable

Scenario: Bridge sentence not in store
  Given the officer completes Step 2.2
  Then the bridge sentence is NOT part of scenarioZeroNarrative in the store
```
